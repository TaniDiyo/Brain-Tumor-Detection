# Brain Tumor Detection Using Deep Learning

## Project Overview

Brain Tumor Detection is a deep learning-based web application that helps identify the presence of brain tumors from MRI scan images. The system leverages a trained Convolutional Neural Network (CNN) model to analyze medical images and provide predictions through an intuitive web interface.

## Key Features

* Upload MRI brain scan images for analysis.
* Detect the presence of brain tumors using a trained deep learning model.
* Generate predictions in real time through a user-friendly interface.
* Integrate machine learning capabilities with a web application for easy accessibility.
* Demonstrate the practical use of Artificial Intelligence in healthcare.

## How It Works

1. The user uploads an MRI brain scan image.
2. The image is preprocessed and prepared for model inference.
3. The CNN model analyzes image patterns and features.
4. The application predicts whether a brain tumor is present.
5. The prediction result is displayed to the user.

## Technologies Used

* **Programming Language:** Python
* **Backend Framework:** Flask
* **Deep Learning Frameworks:** TensorFlow, Keras
* **Frontend:** HTML, CSS, JavaScript
* **Model Architecture:** Convolutional Neural Network (CNN)
